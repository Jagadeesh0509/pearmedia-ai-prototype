## 📋 Typical Workflow

### First Time Setup
```
1. Read: QUICKSTART.md
2. Action: Get API keys (OpenAI + Gemini)
3. Action: Create .env file
4. Action: npm install
5. Action: npm start
6. Test: Try both workflows
```

### Daily Development
```
1. npm start (if not running)
2. Make code changes
3. Test in browser
4. Save and browser auto-refreshes
5. Repeat until happy
```

### Going Live
```
1. Read: DEPLOYMENT.md
2. Choose platform (Vercel recommended)
3. Configure environment variables
4. Deploy
5. Test live site
```

### Assignment Submission
```
1. Read: SUBMISSION.md
2. Verify GitHub repo
3. Verify live website
4. Record execution video
5. Collect all links
6. Submit
```
